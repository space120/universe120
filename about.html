About Universe120

A precise, friendly blueprint playground: draw on a bubble grid with smart snapping, then preview simple 3D walls—right in your browser.

What you can do today

  • 2D editor: Select/Move/Rotate/Group, Line, Pen (polyline), Rectangle, Circle, Arc, Text, Dimension, Erase
  • Smart guides: 0/30/45/60/90° plus equal (≡), parallel (∥), perpendicular (⊥) hints
  • Openings: draw red rectangles inside a room → they become holes in 3D
  • Units & scale bar: choose ft/in/cm/m/etc. and set units per grid step
  • 3D Viewer: orbit & view presets, wall height, wireframe, glTF export
  • 2D export: PNG and SVG for sharing or laser/maker workflows

Quick how-to

  1. Use the Tools ▾ panel under Properties to pick a tool.
  2. Hold Space and drag to pan. Use the mouse wheel to zoom.
  3. Use Opening tool (red) inside a room rectangle to cut window/door holes.
  4. Click 3D Viewer, set Wall height, then Rebuild.
  5. Export with Export PNG/SVG/glTF on the top bar.

Notes

This beta extrudes room outlines directly. Next steps include named door/window components, true wall thickness via offsets, and more editing conveniences.
