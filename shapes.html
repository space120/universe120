Shapes & Lessons

Learn how to build 2D sketches on the bubble grid and turn them into 3D (and glimpse 4D ideas) with Universe120’s builder.

Before You Start

  • Open Universe120 Home (top-left), set Units (cm/in/ft/mi), tune Grid & Dots.
  • Use Pen / Line / Rectangle / Circle to outline your shape on the bubble grid. Snap/Angle guides help you align to 0/30/45/60/90° and intersections.
  • Set Wall height (in your chosen units). Click Build 3D to see the result in the viewer.
  • Export 2D (PNG/SVG) or 3D (glTF) anytime.

Core 3D Shapes (Step-by-Step)

1) Cube / Rectangular Prism

Draw a rectangle; Build 3D → set wall height to make a box.
1

Select Rectangle and draw a box on the grid.

2

Use Dimension if you want exact size (e.g., 300 cm × 300 cm).

3

Set Wall height (e.g., 300 cm). Click Build 3D.

2) Cylinder (Pillar / Tower)

Draw a circle; Build 3D extrudes it upward.
1

Select Circle and pick center + radius (snap to grid).

2

Set Wall height to desired pillar height; Build 3D.

3) Extruded Irregular Room

Use Pen to trace a closed loop. Build 3D extrudes it.
1

Pick Pen. Click around to lay vertices; Enter to finish.

2

Keep angles clean with 30/60/90° lock if you like.

3

Build 3D to extrude the footprint.

4) Openings (Doors / Windows)

Red rectangles fully inside a room become holes in 3D.
1

Draw a room with Rectangle (white stroke).

2

Switch to Rectangle again and draw an Opening (use red stroke via “Wall (px)=0” and keep it inside the room). In the main app, any rect fully inside a room is treated as a hole when you Build 3D.

3

Click Build 3D — the opening cuts through.

5) Cone (Intro)

Circle + guide line. (Future: “Lathe” tool will spin the profile.)
1

Draw a base Circle.

2

Use Line for a center axis up from the circle.

3

(Coming soon) Use Lathe to spin a triangle profile around the axis to make a cone.

6) Sphere (Intro)

Draw a circle + an equator; (future) “Lathe” spins semicircle to a sphere.
1

Draw a Circle and a light ellipse for the equator.

2

(Coming soon) Use Lathe on a half-circle profile to generate a sphere.

4D — Think in Cross-Sections

We can’t “see” 4D directly, but we can learn by slices. A tesseract (4D cube) can be understood by how 3D “slices” change as a 4D hyperplane moves through it — like how a 2D slice through a sphere looks like a circle that grows then shrinks.

Build a Tesseract Projection (Concept)

1

In 2D, draw a small square inside a larger square. Connect corresponding corners (that’s the classic 3D projection of a cube).

2

For 4D, repeat: draw a cube inside a larger cube; connect corresponding corners — that wireframe is a tesseract projection.

3

Use Pen + Angle locks to keep clean parallels. Label faces with Text (Front, Back, Inner, Outer).

4

(Future) “Slice mode” will scrub a 3D slice through the 4D form to show how shapes morph over a 4th axis.

Tips

  • Hold Space to pan; mouse wheel to zoom; Shift+1 to Fit; 1 to reset 100%.
  • Use Angle Lock at 30/60/90° for isometric feels on the bubble grid.
  • Snap range too “sticky”? Tune Snap Range in the Home top bar. Hold Alt to temporarily disable snap.
  • Openings = any Rectangle fully inside a room rectangle (becomes a hole in 3D).
